# 🛒 Fresh Cart — React E-Commerce Web Application

Fresh Cart is a modern, responsive e-commerce web application built using **React** and a powerful set of production-ready tools.
The platform offers a complete shopping experience with authentication, product browsing, cart and wishlist management, checkout, and secure payments.

---

## 🚀 Live Demo

🔗 **[Check out the live demo of Fresh Cart](https://youssefelhabal.github.io/Fresh-Cart/)**

---

## 📌 Features

### 🔐 Authentication

* User **registration** and **login** using **JWT** authentication.
* Protected pages using React Router.
* Form validation powered by **Formik** and **Yup**.

### 🛍️ Shopping Experience

* Explore **Products**, **Categories**, and **Brands**.
* Add items to **Wishlist** or **Cart**.
* Cart includes full **CRUD** functionality:

    * Add items
    * Update item quantity
    * Remove items
    * Clear entire cart

### 💳 Checkout System

* Complete purchases using:

    * **Cash on Delivery**
    * **Online Visa Payment**
* Secure shipping form with real-time validation.

### 👤 User Profile

* View complete **order history**.
* Access saved wishlist and cart.

### ⚡ Performance & UX

* Efficient data fetching and caching with **React Query**.
* Smooth notifications using **React Hot Toast**.
* Offline status detection for better UX.
* Responsive UI built with **Tailwind CSS**.
* Dynamic carousels via **Slick Carousel**.
* Fast development environment using **Vite**.

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/Fresh-Cart.git
cd Fresh-Cart
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Start development server

```bash
npm run dev
```

### 4️⃣ Build for production

```bash
npm run build
```
